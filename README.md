# Cycle Performance Guide

An educational web application that helps optimize training, fueling, and recovery decisions based on menstrual cycle phases. This tool is informed by the SHE Program Master Class led by Jade Skillen.

## Features

- **Phase Selection**: Choose between Menstrual, Follicular, Ovulation, and Luteal phases to view phase-specific guidance
- **Today's Dashboard**: View personalized recommendations for training, fueling, recovery, and things to avoid
- **7-Day Overview**: Interactive timeline showing your cycle phase for the upcoming week
- **Cycle Settings**: Configure your cycle start date, cycle length, and bleed length
- **Mobile-Friendly**: Responsive design that works on all devices

## Getting Started

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to the local development URL (typically `http://localhost:5173`)

### Building for Production

```bash
npm run build
```

## Technology Stack

- **React 19** - UI framework
- **Vite** - Build tool and dev server
- **CSS** - Styling (no CSS framework dependencies)

## Important Disclaimer

This site is an educational, note-style tool intended to support cycle-aware performance decisions and is **not a substitute for professional medical advice**.
